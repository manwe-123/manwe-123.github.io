Materials & Manufacturing Considerations

Material Selection

{% if page.material_selection %} {{ page.material_selection | markdownify }} {% else %}

Steel (AISI 1018/1045): Selected for load-bearing components (shock body, piston rod, end caps) due to excellent strength-to-weight ratio, machinability, and cost-effectiveness. The piston rod is chrome-plated for corrosion resistance and reduced friction.

Bronze (SAE 841): Oil-impregnated sintered bronze selected for bushings due to self-lubricating properties, low friction coefficient against steel, and ability to operate without external lubrication—critical for a sealed shock absorber assembly.

Music Wire (ASTM A228): High-carbon spring steel selected for the compression spring due to high yield strength, excellent fatigue resistance, and consistent spring characteristics under cyclic loading.

{% endif %}

Manufacturing Processes

{% if page.manufacturing_processes %} {{ page.manufacturing_processes | markdownify }} {% else %}
  • CNC Turning: Used for cylindrical parts (shock body, piston rod, bushings) to achieve tight tolerances and smooth surface finishes.
  • CNC Milling: Employed for flat features, keyways, and mounting bracket geometry.
  • Thread Rolling: External threads on piston rod produced via thread rolling for improved strength and surface finish.
  • Centerless Grinding: Applied to piston rod OD for precise diameter control and mirror-like finish.
  • Off-the-Shelf Hardware: Retaining rings, set screws, and O-rings sourced from McMaster-Carr to reduce manufacturing complexity and cost.
{% endif %}

Design Validation & Lessons Learned

Design Validation

✓ Assembly Verification

All 12 components were assembled in SolidWorks with appropriate mates (concentric, coincident, distance). The assembly is fully constrained with 0 degrees of freedom, confirming proper fit and function.

✓ Drawing Accuracy

All 2D drawings were cross-checked against the 3D model using SolidWorks drawing views. Dimensions were verified to match model geometry, and GD&T callouts were validated per ASME Y14.5-2018 standards.

Lessons Learned

{% if page.lessons_learned %} {% for lesson in page.lessons_learned %}
⚠ Challenge: {{ lesson.challenge }}
✓ Resolution: {{ lesson.resolution }}
{% endfor %} {% else %}
⚠ Challenge: Dimensioning the keyway on the piston shaft
✓ Resolution: Researched standard keyway dimensions per ANSI B17.1 and applied appropriate tolerances (+0.002"/-0.000" for width, +0.010"/-0.000" for depth). Added a detailed detail view at 2:1 scale to clearly show the keyway geometry.
⚠ Challenge: Properly specifying GD&T for concentricity between bushing bores
✓ Resolution: Applied positional tolerance (⌀0.005" at MMC) relative to datum axis established by the shock body OD. This ensures both bushings remain coaxial within acceptable limits for smooth piston movement.
⚠ Challenge: Thread callout confusion between UNF and UNC series
✓ Resolution: Verified that 0.75-10 is UNC (coarse) series per ASME B1.1. Updated all drawings to explicitly state "UNC-2A" to avoid manufacturing ambiguity. Added note: "THREADS PER ASME B1.1 UNLESS OTHERWISE SPECIFIED."
{% endif %}
