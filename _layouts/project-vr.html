Testing & Validation

Research Questions

Hypothesis: VR simulation will achieve higher engagement scores than traditional lecture videos while maintaining equivalent learning outcomes.

Variables
  • Independent: Teaching method (VR vs. Video)
  • Dependent: Engagement score (1-10), Quiz score (%)
  • Controlled: Content coverage, time allocation
Procedure
  1. Split class into two groups (A/B testing)
  2. Group A: Traditional video lecture on engineering design
  3. Group B: VR prototyping activity
  4. Both groups complete identical knowledge quiz
  5. Survey engagement on 1-10 scale
Results
  • VR Group: Avg. engagement 8.2/10, Quiz avg. 78%
  • Video Group: Avg. engagement 5.4/10, Quiz avg. 76%
  • Conclusion: Hypothesis supported — higher engagement with equivalent learning

Hypothesis: Medium graphical detail will provide optimal balance between immersion and performance (frame rate) to prevent VR sickness.

Variables
  • Independent: Graphical detail level (Low/Medium/High)
  • Dependent: Frame rate (fps), Motion sickness reports, Quiz score
  • Controlled: Same content, same headset model
Procedure
  1. Three test groups with different detail settings
  2. Monitor frame rates continuously during 65-min session
  3. Track motion sickness reports (Simulator Sickness Questionnaire)
  4. Measure completion time and error count
Results
  • Low: 72fps stable, 0% sickness, lower immersion ratings
  • Medium: 72fps stable, 2% mild discomfort, best overall ratings
  • High: 58-65fps variable, 15% discomfort reports
  • Decision: Locked to Medium detail for deployment

Hypothesis: Mixed-method assessment (quiz + in-game analytics) will provide more comprehensive outcome measurement than either method alone.

Variables
  • Independent: Assessment method (Quiz-only / Analytics-only / Mixed)
  • Dependent: Correlation coefficient, Predictive accuracy
  • Controlled: Same student cohort, same activity
Procedure
  1. Log all in-game actions: time, errors, hints used, objects placed
  2. Administer traditional post-activity quiz
  3. Calculate correlation between analytics metrics and quiz scores
  4. Build predictive model using combined data
Results
  • Quiz-only: Snapshot assessment, r² = 0.45
  • Analytics-only: Continuous data, r² = 0.52
  • Mixed-method: Combined approach, r² = 0.73
  • Conclusion: Mixed-method provides significantly better outcome prediction

Challenge: VR Motion Sickness

Initial high-detail environments caused minor discomfort for some users. Solution: Locked to medium detail, optimized frame rates to stable 72fps, and added comfort vignetting settings during movement. This reduced motion sickness reports by 85%.
